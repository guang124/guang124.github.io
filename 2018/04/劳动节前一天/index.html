劳动节前一天

四月 30, 2018

好困啊，春困秋乏夏打盹，春夏之交就每天都睡不够了。 五一假期，实验室人都出去各玩各的了，甚是无聊，唯有学习。

中午躺床上睡了20多分钟，然后就睡不着了，起来玩起了微信小程序斗地主，玩了三局，金豆输光。其实刚回宿舍的时候也有点困的，正如现在，就是觉得还没玩够，还没到点，迟迟不想去睡，到了一点十分，终于躺下睡了，定了40分钟后的闹钟，模模糊糊了大概20分钟，迟迟未能深眠，便想着把闹钟再加些时间，然而，竟再也睡不着了，躺床上在纠结20分钟，遂起来斗起了地主。 挣扎再三，三点方出门来实验室，继续牛客中级java视频，无奈，脑子甚是混沌，遂打开网易云音乐，视频扔一边，打开我的博客，爱不释手，感慨文章太少，故绞尽脑汁乱编三百，以丰博客。

本来应该结束了，还是想说两句，说啥呢，可能是太过无聊吧，暂时没有屠龙刀，下不去山，心里却不断汹涌着屠龙之心。 却不知屠龙之术已逝。 却不知早已没有龙。 而我也不过是个蜉蝣，却做不到浪荡于瀚瀚宇宙。 现在的我是什么状态呢，我也不知道我。 有所念之人，有念我之人。 而我不知自己为何。 想不透， 早已找不到当年的我， 碌碌为生计， 人生为何。 是这世界太复杂，还是自己复杂了。 可能是自己复杂了吧，随着自己长大成人，在这学校步入社会的大门口。 有些事情需要我来忍耐，有些事情需要承担，有些事情不再那么随心所欲， 早些出去工作吧， 可能一切仅仅是因为， 穷。

可是以前的人们呢，农夫山泉有点田的生活似乎也很快乐，可是现在的社会怎么了，非得一群人拥挤到大城市隐忍的活着才算快乐吗。 有人说是为了后代，你又怎知后代需要什么样的生活呢。 城市，让生活更美好。不可否认，住在大城市十分的方便。 我就是总觉得这个社会的价值观变得有点畸形， 不要忘了，人是从大自然来的， 而现在的人们在离大自然越来越远。

就这些吧，本文可能受网易云音乐中牛奶咖啡的一条动态而感。

学习方法分享【转】

 Aug 8, 2018

在慕课网做了这么多实战课，时间长了，收到了越来越多的同学的问题。除了专业知识上的问题，遇到的最多的问题，就是学习方法方面的问题了。由于我本人不是学习方法的专家，所以不敢轻易说自己的学习方法多么先进。更重要的是，我不认为有一种固定的“好”的学习方法是适合所有同学。如果存在那样的方法的话，学习就太简单了。由于每个人的背景不同，擅长不同，每个人都有适合自己的不同的学习方法；与此同时，不同的领域，由于其本质不同，学习方法也应该是不同的。我坚信：每个人去寻找适合于自己的学习方法，是每个人一生的话题。别人叙述的所谓的“学习方法”，只能作为参考用。

不过我在慕课网做答疑的过程中，确实看到了很多同学的学习方法，有改进的空间。在这里，我想简单总结一些我见过的，认为可能不是那么得当的学习方法，仅供参考。这篇文章不是一套“如何学习”的理论，而是一些零碎的个人见解。更像是在总结“学习中的那些坑”，不过，在我写完这篇文章之后，回头看，我觉得，我自己的高效学习的秘诀，其实都在这篇文章里了。

Hexo中的Yelee主题，首页不显示文章

 Apr 4, 2018
最近在用Hexo搭建博客，刚刚更换了Yelee主题，在搭建完成后发现了一个BUG，在所有文章列表中有文章，首页左面正常显示，但是首页右面却没有显示文章。 解决方法是把yelee文件夹下的_config.yml文件修改一下，如下，大概是在234行，在onload前面加一个“#”，注释掉这一行后就能正常运行了。 可以点击这里查看我博客上面的效果。 search: #on: true #onload: false #这里有更改，把on前面的‘#’去掉 ## true: get search.xml file when the page has loaded ## false: get the file when search box gets focus
